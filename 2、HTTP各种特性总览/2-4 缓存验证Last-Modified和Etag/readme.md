## 1、缓存验证头
### 1.1 使用条件
> 当设置了```Cache-Control:no-cache```时，需要进行缓存验证。
### 1.2 缓存验证设置方法有```两种```
- ```Last-Modified```
  * 上次修改时间
  * 配合```If-Modified-Since```（常用）和```If-Unmodified-Since```使用
  * 对比上次资源修改时间来验证资源是否要更新
- ```Etag```（更加严格的验证）
  * 数据签名（只要有任何的数据修改都会更改Etag，比如对内容进行计算得到hash）
  * 配合```If-Match```和```If-No-Match```使用
  * 对比资源签名判断是否使用缓存
